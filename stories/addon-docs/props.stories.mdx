## Controls Story

<Preview>
  <Story
    name="Controls"
    args={{
      background: '#ff0',
      label: 'apple',
      count: 5,
    }}
    argTypes={{
      count: { control: { type: 'range', min: 0, max: 10 } },

      return (
        <ButtonGroup background={background}>
          {range(count).map((i) => (

        </ButtonGroup>
